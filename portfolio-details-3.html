Wellness.ly, a Hackathon project

Inspiration :: We were inspired by people we know ourselves, working women in our families. They take up so many responsibilities and have all but little to no time to look after themselves. We have wellness.ly to act as an accessible, all-in-one self-care website(soon to be an app) by working women who want a quick hack for self-care. We saw that mental health, skin, and sleep were compromised the most. So we created solutions for them.

What it does :: Mental Health API: Positivity Pal, a mental health chatbot that recognizes over 23 emotions and supports you emotionally. Skin-Care and Makeup Help: Working women's everyday routine doesn't go without makeup. We created personalized skincare routine recommendations based on skin type, preference, and time & money constraints. Currently, we help with natural skincare, professional look, minimal looks, etc. Sleep Tracking: A data-visualization algorithm that plots your sleep goal and how much you slept according to your own inputs. It shows how you have done over a time period. We hope to make it even more customized and helpful in the future by adding a personalized sleep routine feature.

How we built it :: The mental health API was created using dialogflow--we trained it to recognize emotions and intents from text using NLP. We added training phrases that it would respond with as well. This was all done on dialogflow and deployed on google cloud. The skincare and makeup help was created with bubble.io workflow. We did personalized recommendations using machine learning algorithms on streamlit coded with python. We embedded that on bubble.io. It still has a long way to go. The sleep tracker was made on bubble using HTML code to make the graph. The graph was made to gather dynamic input from bubble itself.

Accomplishments that we're proud of:: While trying to make the graph, we couldn't figure out how to make it with dynamic data, in other words, customized data. We eventually did found a solution. We were able to make an all-in-one self-care hack for such a big community, working women. We were able to not only make just one type of api, but three whole functionalities together.

What we learned:: Creating a game using a software called Ren'Py (which uses python and its own 'language'). For the website we used bootstrap and general html and css.

What's next for Wellness.ly:: A whole new personalized sleep tracking system that's novel and easy to use. Also we keep on bettering our skin care and makeup system.

Built With:: bubble.io dialogflow domain.com google-cloud python streamlit
